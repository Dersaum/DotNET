# .NET

## dotnet new
Cria um novo projeto, arquivo de configuração ou solução com base no modelo especificado.
> dotnet new console -n "nome do Aplicativo"


#### <br>*Extensões do VS CODE sugeridas para serem instaladas*
- *C#*<br>
- *C# Extensions*<br>
- *vscode-icons*<br>
